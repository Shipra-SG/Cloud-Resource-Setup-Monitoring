# Cloud Resource Setup and Monitoring

## Overview
This project demonstrates foundational cloud skills, including:
- Launching and configuring AWS EC2 instances
- Creating S3 buckets and uploading files
- Managing IAM users and roles
- Monitoring EC2 health via CloudWatch
- Documenting SOPs and troubleshooting steps

## Repository Structure
```

cloud-resource-setup-monitoring/
├── docs/         # SOPs and screenshots
├── demo/         # Sample files for testing
└── README.md

```

## Steps
1. Launch EC2 instance using `docs/SOP_EC2_Setup.pdf`
2. Create S3 bucket using `docs/SOP_S3_Setup.pdf`
3. Add IAM users and roles using `docs/IAM_User_Guide.pdf`
4. Monitor EC2 health and alerts via CloudWatch
5. Upload test files to S3 using `demo/test.txt`
